Surveyscape Privacy Policy

Surveyscape is a utility application created to provide support to research studies. Below is our Privacy Policy explaining how data is collected and used. The below clauses are separated into two sections, one for a specified user, and one for a non-specified user.

A specified user is the primary user for Surveyscape. Due to the nature of this application, we only collect the data from users who have previously agreed to the terms of a survey through an external survey organizer on our end. These specified users are given a five digit number to use to log into Surveyscape. The user then has access to an external Qualtrics survey that is created by our survey organizers.

A non-specified user is for those who do not wish to participate in an organized survey, but would still like to test out some functions of the app. We do not store any information from these users.

If at any point you believe that Surveyscape has collected data without your consent or has broken any of these privacy clauses, please promptly contact your survey organizer.

Last Updated February 28, 2021

1. What Data does Surveyscape Collect?

1.1 For Specified Users: Surveyscape only collects location and time data to perform necessary functions. Surveyscape will continuously monitor precise location locally but will only externally store data relevant to the purpose of the chosen study. Specifically, Surveyscape will use a hardcoded coordinate location to only track when a user enters or exits a specific radius around these coordinate points. The date and time of entry and exit of the given location will be collected, as well as the duration of time spent within the region. The collection of this data will be continuous and automatic for the duration of the study, or a duration that a user has previously agreed to with our external survey organizer. The collection of this data is private and is only attached to the five-digit number provided prior to the download of Surveyscape. There is no identification data required for the use of Surveyscape.

1.2 For Non-Specified Users: Surveyscape only collects location data to perform necessary functions. Surveyscape will continuously monitor precise location locally and will not store any information externally. Surveyscape will set a radius around a user-selected coordinate point, and track when a user enters or exits a specific radius around these coordinate points. The collection of this data will be continuous and automatic until the user ‘logs out’ of the non-specified user side of the application.

2. How is This Data Collected?

2.1 For Specified Users: Users must voluntarily grant Surveyscape permissions to collect all data. Surveyscape will prompt a user upon opening the app to collect location data. Due to the nature of the app, Surveyscape will likely require the user to ‘always’ provide location data. Location data will be collected automatically and locally through the application. A user’s precise location or any precise location data will NEVER be stored anywhere at any time and is only used to record time data relevant to an entry or exit of a specified region corresponding to the organized survey. Time data will be collected from the user’s local phone time. Surveyscape does NOT store location data, only time data corresponding to the location of a user. The collection of this data is private and is only attached to the five-digit number provided prior to the download of Surveyscape. There is no identification data required or collected by Surveyscape.

2.2 For Non-Specified Users: Users must voluntarily grant Surveyscape permissions to collect all data. Surveyscape will prompt a user upon opening the app to collect location data. Due to the nature of the app, Surveyscape will likely require the user to ‘always’ provide location data. Location data will be collected automatically and locally through the application. A user’s precise location will NEVER be stored anywhere at any time and is only used to send notifications relevant to an entry or exit of a specified region corresponding to the user selected location.

3. How is This Data Used?

3.1 For Specified Users: Surveyscape is a utility application used to aid in research studies. Time data relevant to when a user enters and exits a specified region are collected into a Google Firebase Database, which a survey organizer will then analyze for specific research. A user will not receive an authenticated log in number without accepting the terms provided both here and by the survey organizer prior to the download of Surveyscape.

3.2 For Non-Specified Users: No data from non-specified users is stored or used externally after it is collected locally from the application.

4. Use of Third Party Services

4.1 For Specified Users: Surveyscape utilizes a Google Firebase Database to store time data relevant to the specific study. Surveyscape does NOT use any third party service or application to record or store user location data for specified users. No local or external services are used by Surveyscape to collect identifiable information.

4.2 For Non-Specified Users: Surveyscape does not use any third party services to record, store, or collect any user data for non-specified users.

5. Retention of Data

5.1 For Specified Users: Surveyscape retains the collected data relevant to a study only for the duration of the study. Due to the variability of the duration of a study, this information is provided to a specified user prior to receiving a authenticated log in number. The user has the option to accept or reject the terms with the survey organizer and will only become a specified user if the user voluntarily accepts. Once a specified user of Surveyscape, the recorded time data will be stored in a Google Firebase Database until the survey duration has completed, or the user withdraws consent.

5.2 For Non-Specified Users: Surveyscape never retains any data from non-specified users.

6. Refusal or Withdrawal of Consent

6.1 For Specified Users: Surveyscape will request permissions to collect location data from the users’ phone. If, at any time, a user refuses or wants to withdraw allowance to collect this data, they may follow the following instructions to withdraw these permissions: Go to Settings > Surveyscape > Location > Set Allow Location Access to “Never”. If a user requests to withdraw from an ongoing study altogether, they may follow these instructions and then promptly contact the survey organizers about their withdrawal.

6.2 For Non-Specified Users: Surveyscape will request permissions to collect location data from the users’ phone. If, at any time, a user refuses or wants to withdraw allowance to collect this data, they may follow the following instructions to withdraw these permissions: Go to Settings > Surveyscape > Location > Set Allow Location Access to “Never”.

7. What the Collected Data is NOT Used For

7.1 Surveyscape does NOT collect or connect any direct identifiers to the data collected. For example, Surveyscape does not require a user to input their name, email address, phone number, address, or any other personally sensitive identifiable data connecting data to a user’s identity.

7.2 Specific or precise location data is NOT stored by Surveyscape externally and does not have the capability to do so.

7.3 Time data that is stored is viewed only by professional data analytic researchers and does not have any identifiable data. A researcher who will view the collected data only has access to the information a user has consented to, which is unidentifiable time data connected with the entry and exit of a specified location.

7.4 “Personal Data” and “Personal Information” is defined under relevant privacy laws as data linked to the user. Surveyscape does not collect any data that can be linked back to a user’s identity.
